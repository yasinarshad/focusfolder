# Changelog

All notable changes to the "Focus Folder" extension will be documented in this file.

## [1.3.0] - 2025-01-17

### Added
- Full file operations (cut, copy, paste, rename, delete)
- Folder customization (colors, icons)
- Keyboard shortcuts for all operations
- Integration with Yasin Favorites extension
- Move to Folder command
- Create Templated Folder command

### Improved
- Better drag and drop reordering
- Enhanced context menus

## [1.0.0] - 2024-12-01

### Added
- Initial release
- Focus on multiple folders simultaneously
- Dedicated FOCUSED panel in Explorer
- Basic add/remove focus operations
- Sort options (A-Z, Z-A, Modified, Manual)
